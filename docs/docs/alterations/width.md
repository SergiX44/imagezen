# `width()`

```php
->width(): int
```
Get the image width.

## Parameters

<i>This method has no parameters.</i>

## Returns

`int`: The image width

## Example

```php
use SergiX44\ImageZen\Image;

$width = Image::make('path/to/image.jpg')->width();

```
