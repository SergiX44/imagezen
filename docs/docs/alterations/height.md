# `height()`

```php
->height(): int
```
Get the image height.

## Parameters

<i>This method has no parameters.</i>

## Returns

`int`: The image height

## Example

```php
use SergiX44\ImageZen\Image;

$height = Image::make('path/to/image.jpg')->height();

```
